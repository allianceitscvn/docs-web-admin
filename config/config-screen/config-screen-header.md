# Config Screen Header

## Tóm tắt

```
{
    "Header":{
        "show":true,
    }
}
```

## Chi tiết

| Name | DefaultValue | Mô tả                    |
| ---- | ------------ | ------------------------ |
| show | true         | có show header hay không |
|      |              |                          |
|      |              |                          |
